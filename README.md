# Calculator-for-beginners

So the initial thought is to create a calculator, not the scientific by the ordinary one. The kind that even a 2 year old can use and understand.
This is a fun project that helps us in undersatnding how apps or programmes are setup together to make one functional programme, where in tthis case the programme will
be a  calculator

The logic on for this calculator is for it to be able to do the following :

1. Give out correct and accurate results
2. Add, Subtract, Divide and Multiply
3. Be able to clear out(erase when requested or when an erase button is clicked)
4. Have a stylist colour vibe going on
